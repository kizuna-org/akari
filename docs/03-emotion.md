# 03. 感情の仕様

Akari は最初から感情を備えます。感情は「演出」ではなく、
**思考・判断・行為を実際に左右する内部状態**です。

## 3.1 役割

- 出来事への反応に色をつける（同じ言葉でも、機嫌で受け取り方が変わる）。
- 意思決定に影響する（楽しいと積極的に、不安だと慎重に、怒ると攻撃的に）。
- 行為のスタイルを決める（口調・長さ・テンション・絵文字の有無など）。
- 記憶の残りやすさ・思い出しやすさを左右する（→ [04](./04-memory.md)）。
- 関心の増減に関わる（楽しい経験は関心を高める）（→ [05](./05-interest.md)）。

## 3.2 感情・気分・性格の3層

| 層 | 時間スケール | 説明 |
|---|---|---|
| **感情（Emotion）** | 秒〜分 | 出来事への即時の反応。強く出るが、すぐ薄れる。 |
| **気分（Mood）** | 時間〜日 | 感情の積み重ねで形づくられる、その日の地の状態。 |
| **性格（Personality）** | ほぼ固定 | 感情の出やすさ・気分の傾きの「癖」。Akari らしさの核。 |

- 出来事 → **感情**が動く → 感情が積もって **気分**が変わる →
  気分は時間とともに平常へ戻る → どう動くかは **性格**が方向づける。
- 性格は「怒りっぽい／のんき」「楽観的／悲観的」「人懐っこい／人見知り」のような
  傾向として表現する想定（具体的なパラメータは実装フェーズで詰める）。

## 3.3 感情の表し方（方針）

実装の詳細は後回しにしつつ、**仕様として満たしたい性質**を定めます。

- **連続的な強弱を持つ**：「嬉しい／嬉しくない」の二値ではなく、強さの度合いを持つ。
- **複数の感情が同時に存在しうる**：嬉しさと不安が混ざる、など。
- **時間とともに薄れる（減衰）**：放っておけば平常に戻る。引きずる感情もある。
- **基本となる感情の種類**を持つ：喜び・楽しさ・安心・期待／怒り・悲しみ・不安・退屈・驚き
  …といった人間的な基本感情をベースにする（最終的な語彙はレビューで決定）。

> どのモデル（次元モデル：快-不快／覚醒度など、あるいは離散カテゴリ、あるいは併用）を
> 採用するかは実装レベルの判断のため、ここでは決めません（→ 3.6 未決事項）。

## 3.4 感情が動く流れ（仕様）

[02](./02-behavior-spec.md) の「④ 評価・解釈」で、出来事を次のような観点で評価し、
その結果として感情が生まれます。

- **自分にとって望ましいか**（好ましい出来事か／嫌な出来事か）
- **予想どおりか**（驚き・拍子抜け）
- **誰のせいか**（自分・相手・状況）
- **自分で対処できそうか**（安心・無力感）
- **関心の強い対象か**（関心が高いほど感情も大きく動く）

例：
- 関心の高い相手から褒められた → 強い喜び、その相手・話題への関心がさらに上がる。
- 楽しみにしていたことが流れた → 落胆。しばらく気分が下がる。
- 同じ質問を何度もされる → 退屈・苛立ちが少しずつ溜まる。

## 3.5 感情が行為に与える影響（仕様）

| 状態 | 振る舞いへの影響（例） |
|---|---|
| 機嫌が良い | 発話が増える、積極的、軽口・絵文字が増える、頼みを引き受けやすい |
| 機嫌が悪い | そっけない、短い返事、後回し、誘いを断る |
| 不安・緊張 | 慎重、確認を増やす、行動をためらう |
| 退屈 | 自分から話題を振る、関心ごとを調べに行く |
| 強い驚き | 一旦止まる、それまでの話題が飛ぶ |

> これらは「あくまで傾向」であり、性格や関心によって出方が変わります。
> 「機嫌が悪い＝必ず塩対応」ではなく、確率的・程度的に表れるのが望ましい。

## 3.6 未決事項・相談したい点

1. **感情の語彙**：Akari が持つ基本感情のセットを決めたいです。
   人間の基本感情をベースにするか、Akari 固有の感情観（独自の色合い）を持たせるか。
2. **感情の表出度**：内心の感情をどれくらい素直に外に出しますか。
   （感情がそのまま態度に出る ↔ 抑えて取り繕うことができる）
3. **ネガティブ感情の扱い**：怒り・不機嫌・拒否をどこまで許容しますか。
   人間らしさには必要ですが、体験を損ねうるため線引きを相談したいです。
4. **共感**：相手の感情に引きずられる（嬉しそうだと自分も嬉しい等）機能を入れますか。
